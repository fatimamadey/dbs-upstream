BREAKING: Interlock Ransomware Exploits Cisco FMC Zero-Day for Root Access

LIVE COVERAGE
The Hacker News — Security Desk LAST UPDATED: 2 MIN AGO

A critical CVSS 10.0 vulnerability in Cisco Secure Firewall Management Center allows unauthenticated remote attackers to bypass authentication and execute arbitrary code as root via insecure deserialization of Java byte streams.

The Interlock ransomware group is actively exploiting this zero-day to gain persistent root access to enterprise firewall infrastructure. Security researchers warn that the insecure deserialization flaw in the Java-based management interface requires no authentication, making it trivially exploitable from the network. Organizations running Cisco Secure Firewall Management Center are urged to apply emergency mitigations immediately while awaiting an official patch. CISA has added this vulnerability to its Known Exploited Vulnerabilities catalog.
